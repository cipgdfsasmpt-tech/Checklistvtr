ARQUIVOS:
  index.html           → Tela inicial (escolha do checklist)
  checklist-4rodas.html → Checklist Viatura Automotiva
  checklist-2rodas.html → Checklist Motocicleta
  manifest.json        → Configuração do PWA
  sw.js                → Service Worker (offline)
  icon-192.png         → Ícone do app
  icon-512.png         → Ícone do app (alta resolução)

COMO PUBLICAR (gratuito, sem loja):
  1. Acesse https://app.netlify.com/drop
  2. Arraste toda esta pasta para a página
  3. Copie o link gerado (ex: https://meu-app.netlify.app)
  4. Envie o link para os militares

COMO INSTALAR NO CELULAR:
  Android (Chrome):
    → Abra o link no Chrome
    → Toque no banner "Adicionar à tela inicial"
    → ou Menu (⋮) → "Adicionar à tela inicial"

  iPhone (Safari):
    → Abra o link no Safari (obrigatório)
    → Toque em Compartilhar (⬆)
    → "Adicionar à Tela de Início"
    → Toque em "Adicionar"

FUNCIONA OFFLINE:
  Após instalar, o app funciona sem internet.
  O envio para WhatsApp e Google Sheets requer conexão.
